## Requirements
- [Yarn](http://yarnpkg.com/)
- [ESLint](http://eslint.org)
- [EditorConfig](http://editorconfig.org/)

## Getting Started
```
git clone git@github.com:MathieuDebit/starter-pack.git <project-name>
cd <project-name>
yarn install
yarn start
yarn build
```
